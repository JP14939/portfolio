02 / 04
AI Video Production Pipeline

An end-to-end content automation system for TikTok. Pulls raw footage from Google Drive, asks the user to validate quality, then uses computer vision to reverse-engineer approved clips into scene and motion prompts. Those prompts feed into Google Veo 3 to generate a matching AI clip. Both videos are assembled dynamically in FFmpeg — consistent format, auto-captions, audio, and text overlays — outputting a ready-to-post 60-second video.

Algorithm Complexity Analyser

A developer tool that takes two code snippets and runs a full comparative analysis: static checks for comment quality, nesting depth, and function size; dynamic profiling for peak memory usage; Big O complexity estimation. Results are plotted on a comparison graph. Feedback is written by a locally-run Ollama model to keep costs at zero. Comes with a localhost frontend for clean usability.

Be Present — Life OS Mobile App

A mobile productivity app built around emotional accountability. The core mechanic is a persistent attachment object with a live health score — it deteriorates in real time when you leave a committed deep work session to scroll or switch apps. Features a dashboard with work hours tracked over time, a configurable boredom pre-session (user-set stillness window before starting), and iOS background detection to enforce session integrity.
